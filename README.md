# fitzoneFitness
<h1>FitZone Fitness Website</h1>
    <p><strong>FitZone Fitness</strong> is a web platform designed for a fitness center that allows customers to manage their memberships, join classes, view schedules, provide feedback, and interact with staff through messages. It includes login and registration functionalities, class joining features, and a blog section with fitness tips, workout routines, and healthy recipes.</p>
    <h2>Features</h2>
    <ul>
        <li><strong>Login/Registration</strong>: Customers, staff, and admins can register and log in to their respective accounts.</li>
        <li><strong>Class Management</strong>: Customers can view available classes and join them.</li>
        <li><strong>Message System</strong>: Staff and admin can send messages to customers.</li>
        <li><strong>Feedback System</strong>: Customers can submit feedback, and admins can respond to them.</li>
        <li><strong>Blogs</strong>: A section for fitness-related blogs, workout routines, and healthy recipes.</li>
        <li><strong>About Us</strong>: Information about FitZone Fitness Center.</li>
    </ul>
